# Design System Document

This document outlines the core aesthetic and functional principles of our design system.

## Brand Identity

Our brand identity is built on a foundation of **light mode** aesthetics, emphasizing clarity and approachability.

### Color Palette

The primary brand color is `#4A44E4`, a vibrant and engaging hue used for key interactive elements and calls to action. We use `#2DD4BF` as a tertiary accent color, providing a refreshing contrast for highlights and decorative elements.

### Typography

*   **Headlines** utilize `plusJakartaSans`, a modern and clean typeface that ensures strong visual hierarchy.
*   **Body text** and **labels** are set in `manrope`, chosen for its excellent readability and versatility across various content types.

## UI Elements

### Shape and Form

UI elements feature a **moderate roundedness (2)**, striking a balance between modern softness and professional structure.

### Spacing

We implement **normal spacing (2)**, which provides a comfortable and balanced amount of whitespace, ensuring content is easy to digest without feeling overly dense or sparse.
